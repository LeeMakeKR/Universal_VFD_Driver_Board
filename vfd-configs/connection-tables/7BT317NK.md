## 연결 요약
- **총 핀 수**: 28개 (G0~G6: 7개, P0~P20: 21개)
- **MAX6921 #1**: OUT0-OUT19 (Grid G0-G6 + Segment P0-P12)
- **MAX6921 #2**: OUT0-OUT7 (Segment P13-P20)

## 제어 방식
1. Grid 스캔: G0~G6를 순차적으로 활성화
2. Segment 제어: 각 Grid가 활성화될 때 해당하는 Segment들을 제어
3. 듀티 사이클: 밝기 조절
